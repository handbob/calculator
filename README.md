# Interactive commandLine calculator

### Requirements
[Java](https://adoptium.net), [Gradle](https://gradle.org)

### Build & Run project

#### Run
* Windows, MacOS/Linux
```
gradle build
```
#### Build
* Windows, MacOS/Linux
```
gradle build
```
#### Run Jar file
* Windows, MacOS/Linux
```
java -jar calculator-1.0.0.jar
```
